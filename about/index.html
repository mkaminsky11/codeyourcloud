Todo List

In order to use the todo list, use the 3 keywords

        //NOTE: a note
        //TODO: a todo item
        //FIXME: help!

Each of these will result in an item for the todo list. Once the item is checked off, it, along with the note, disappears. For example, if the first "NOTE" is clicked, the content will be changed to:

        //
        //TODO: a todo item
        //FIXME: help!

Vim

For users who enjoy using the vim code editor, Code Your Cloud has a very basic vim mode. So far, only the basic modes (visual, insert, etc.), arrow keys, and save (:w) are functional. Look for more additions in the future.

Preferences

The supported preferences are:

  • Theme
  • Font size
  • Line wrap
  • Line Numbers
  • Vim mode
