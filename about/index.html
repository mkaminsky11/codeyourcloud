Code Your Cloud is on a mission
 

Code Your Cloud is an in-browser collaborative code editor for Google Drive, which combines the capabilities of Google's cloud services with the functionality of CodeMirror syntax highlighting and editing.


This project was born out of a deep frustration for the lack of competent code editors for Google Drive. Although some code editors (namely Editey and Drive Notepad) do exist, they do not provide the functionality, polish, and usefullness required for many projects. This is an attempt to fix that.
